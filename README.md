# Spotify Home Clone

A simple **Spotify Home Page Clone** built using only **HTML and CSS**. This project replicates the look of Spotify's homepage without any functionality.

## Features
- Home page layout
- Styled with CSS
- Responsive design for different screen sizes

## Tech Used
- HTML
- CSS
